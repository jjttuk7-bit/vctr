## 5. Logo

### Logo Principle

```
V    → font-weight: 900  (black — precision, direction)
ctr  → font-weight: 200  (extralight — speed, clarity)

The weight contrast is the identity. Equal weights = identity lost.
V is always Vctr Indigo (#6366F1). ctr changes per background.
```

### TSX Implementation

```tsx
// components/Logo.tsx
export default function Logo({ variant = 'default', size = 'sm' }: Props) {
  const isDark   = variant !== 'white'
  const bg       = isDark ? 'bg-vctr-black' : 'bg-vctr-surface border border-slate-200'
  const ctrColor = isDark ? 'text-white' : 'text-vctr-black'

  return (
    <Link href="/" className={`inline-flex items-baseline rounded-xl ${sizes[size]} ${bg}`}>
      <span className={`font-black text-vctr-indigo tracking-tight`}>V</span>
      <span className={`font-extralight ${ctrColor} tracking-tight`}>ctr</span>
    </Link>
  )
}
```

### Logo Rules

```
✅ Correct
- Near Black background + Indigo V + White ctr
- Surface background + Indigo V + Near Black ctr
- Minimum size: 20px (mobile)

❌ Prohibited
- Arbitrary background colors
- V and ctr at equal font weights
- Any color for V other than Vctr Indigo
- Stretching or distorting the logo
```
